## Download project
```
git clone https://github.com/is-tech-y24-1/Do-Svyazi.Music.git
```

## Restore dependencies
```
dotnet restore
```

## Start project
```
dotnet run --project Server/DS.Music.WebApi
```

## Run unit tests
```
dotnet test --no-build
```
